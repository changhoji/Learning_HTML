녹차맛 HTML

태그 목록

태그 - table

개요

표를 나타내는 태그입니다.

thead - th, tbody - td, tfoot - td 태그가 존재합니다.

태그 - td

table data의 약자로 table의 각 항을 나타내는 태그입니다.

태그 - tr

table row의 약자로 table의 각 행을 나타내는 태그입니다. td태그 항목 여러개를 tr태그로 묶습니다.

예시

코드

<table border="1">
    <tr>
        <td>직업</td> 
        <td>지역</td> 
        <td>분류</td> 
        <td>시너지</td>
    </tr>
    <tr>
        <td>건슬링어</td> 
        <td>아르데타인</td> 
        <td rowspan="2">딜러</td> 
        <td>상시치적 10%</td>
    </tr>
    <tr>
        <td>창술사</td> 
        <td>애니츠</td> 
        <td>순간치적 18%</td>
    </tr>
    <tr>
        <td>바드</td> 
        <td>실린</td> 
        <td>서포터</td> 
        <td>상시 주는피해증가 10%</td>
    </tr>
</table>

결과

직업 지역 분류 시너지
건슬링어 아르데타인 딜러 상시치적 10%
창술사 애니츠 순간치적 18%
바드 실린 서포터 상시 주는피해증가 10%
